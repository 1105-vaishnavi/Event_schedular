Event Scheduler System (Flask)

A web-based application built using Flask that allows organisations to schedule events, allocate shared resources, and detect conflicts efficiently. The system ensures that resources are not double-booked and provides a resource utilisation report within a selected time range.

Features

User Authentication (Signup / Login / Logout)

Create and manage events (workshops, seminars, classes)

Add and manage shared resources (rooms, instructors, equipment)

Allocate resources to events

Conflict detection for overlapping resource usage

Resource utilisation report with total hours and upcoming bookings

Clean dashboard UI with sidebar navigation

Protected routes using login authentication

Tech Stack

Backend: Python, Flask

Database: SQLite

ORM: SQLAlchemy

Frontend: HTML, CSS (Jinja2 templates)

Authentication: Flask session-based login

Version Control: Git & GitHub
